# odin-recipes

## Project: Recipes

This is my first project in TOP.

I created a simple website and webpages following the instructions
in the Foundations course.

*Below* is the summary of what I've done to perform the project.

1. In GitHub, I created a new repository named **odin-recipes** then enabled
the README.md file. Before going to the terminal, I copied the SSH key.
2. In terminal, that newly created repo was cloned using the command
`git clone paste-your-ssh-URL-here`
3. Next, I created the necessary HTML files starting with **index.html**
in the root project folder. The relative links were store in the *recipes*
folder. Take note that I utilized the terminal to create the files to apply
what I learned previously. I used the command *touch*.
4. While creating the HTML files, I also searched and downloaded free
images related to the recipes and stored them in the *img* folder.
5. After checking the pages if already working good, I went back to the
terminal. I staged the new files then committed them. I applied what I
learned about having a good commit in the message tab.
6. Lastly, I updated this README file to make a summary of what I've learned
and applied to make this project.
